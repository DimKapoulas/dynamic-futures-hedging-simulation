# Dynamic Derivatives Hedging Simulation (Python)

This project features a quantitative simulation framework that demonstrates delta-hedging strategies using futures contracts to minimize portfolio risk. 

### Key Features:
* **Asset Path Generation:** Modeled underlying asset prices using Geometric Brownian Motion (GBM).
* **Hedging Strategy:** Implemented dynamic rebalancing intervals to evaluate delta-neutral positioning.
* **Analysis:** Visualized PnL variance reduction and tracked tracking errors relative to rebalancing frequency.
